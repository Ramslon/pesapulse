# PesaPulse 💰

PesaPulse is a full-stack expense tracking mobile application built using Flutter and Laravel API.

## 🚀 Features

- User Authentication (Laravel Sanctum)
- Login & Registration
- Expense Management
- Add / Edit / Delete Expenses
- Expense Search
- Analytics Dashboard
- Category Pie Charts
- User Profile Management
- Bottom Navigation
- Auto Login with SharedPreferences
- Dark Mode Ready Architecture
- Responsive UI/UX

---

## 🛠 Tech Stack

### Frontend
- Flutter
- Dart

### Backend
- Laravel
- MySQL
- Sanctum Authentication

## ⚙️ Installation

### Backend

```bash
git clone <repo-url>

cd pesapulse

composer install

cp .env.example .env

php artisan key:generate

php artisan migrate

php artisan serve
```

### Frontend

```bash
cd pesapulse_mobile

flutter pub get

flutter run
```
